# 🇸🇬 新加坡找工作终极攻略 🚀

哈喽，各位在新加坡打拼或者准备来新加坡闯荡的小伙伴们！最近刷小红书，发现好多关于在坡县找工作的讨论，感觉"太难了"简直是大家的口头禅。别慌！这篇攻略汇总了超多来自小红书用户的真实血泪经验和成功秘籍，希望能帮你在这场"战斗"中少走弯路，早日上岸！💪

## 📄 第一章：认清现实 —— 当前新加坡就业市场概览

正所谓知己知彼，百战不殆。在投简历之前，我们先来看看现在新加坡的就业市场到底是个什么情况。

### 📉 **普遍挑战：僧多粥少，压力山大**

- **竞争白热化**：一个职位几百人申请是常态，有经验的本地人都感叹"工作机会少了"。
- **HC 收紧**：不少公司，尤其是科技和互联网行业，都在缩减招聘名额 (HC)，甚至出现"招人招了一两年都没招到"的"幽灵岗"。
- **期望管理**：做好心理准备，HR 可能会和你"谈心"，暗示你"要求薪资太高了"，甚至有公司直接要求"cut pay 20%"。
- **杳无音讯**：简历石沉大海，面试后没下文，甚至拿到口头 offer 最后黄了的情况也时有发生。别太往心里去，这不是你一个人的遭遇。

### 👑 **"身份"是王道：PR/公民 VS 外国人准证**

这是无数遍被提及的痛点，也是无法回避的现实！

- **PR/公民优先**：很多公司（尤其是好职位和大厂）会优先考虑新加坡公民 (Citizen) 和永久居民 (PR)。HR 拿到简历的第一句话往往是："**这个候选人有身份吗？**"
- **准证之困 (EP, SP, WP)**：对于需要公司申请工作准证（Employment Pass, S Pass, Work Permit）的外国人来说，难度直线上升。公司有名额 (Quota) 限制，申请流程也更繁琐，导致很多雇主不愿意招外国人。
  - "面试了两个都成功了，但是不够人头弄不到准证，裂开[笑哭 R]"
  - "我公司招人哦...但很多大公司的管培岗位只要本地人。" - 来自猎头的声音
- **MOM 潜规则**：即使 MOM（人力部）报告说就业市场"稳中向好"，但实际情况可能是另一回事。有评论直言："你竟然会信 MOM 这个玩意儿"。

### 🎓 **学历与经验：本地经验更吃香**

- **本地经验 > 一切**：新加坡职场非常务实，**本地工作经验**往往比一纸文凭更受青睐。
- **海外名校光环褪色**：国内顶尖名校如"清北"的毕业生，如果没有本地经验或身份加持，在新加坡的认可度可能不如预期。"Top2 在国外就别端着架子了，这边没身份都一样。"
- **本地学历有优势，但非绝对**：NUS, NTU, SMU 等本地大学毕业生相对有优势，但也面临激烈竞争。"我身边好多 NUS，NTU，SMU 新加坡本地的大学毕业生...也面临着...还没有找到工作的情况。"
- **硕博可能还不如本科/Poly**：对于一些职位，雇主认为本科甚至 Poly（理工学院）毕业生就能胜任，不会为硕士或博士学历支付更高薪水。"新加坡公司太务实，可以本科干的活，不会多付一个子找硕博。"

## 🛠️ 第二章：装备升级 —— 求职核心策略与技巧

了解了"战场"形势，接下来就要升级我们的"装备"了！

### 🚀 **LinkedIn – 你的首要战场！(被多位上岸者强推！)**

这绝对是**重点中的重点**！很多成功上岸的小伙伴都把 LinkedIn 列为最高效的渠道。

- **为什么是 LinkedIn?**
  - 一位成功上岸的朋友分享："最终，还是通过 LinkedIn 找到了现在的工作。" 另一位也提到："成功率：LinkedIn >> jobstreet"。
- **优化你的 LinkedIn 档案 (Profile is Everything!)**
  - **英文名**：方便外国人辨认和搜索。
  - **专业头像 (Professional Photo)**：第一印象很重要。
  - **Banner 图**：精心设计一下，放上你的职业关键词，让人一眼就知道你是做什么的。
  - **个人简介 (Summary/About)**：清晰、简洁地展示你的核心技能、工作经验和求职意向，堆满关键词！
  - **案例分享**："在使用 LinkedIn 时一定要注意，先打理好你的账号。这点非常重要！...让人一点开就清晰地知道你是做什么的，想找什么工作。"
- **积极拓展人脉 (Connections Matter!)**
  - **目标 100+ Connections**：很多评论提到，老外认为至少有 100 个以上的连接，才会觉得你是个"真实存在的人"，增加账户可信度。
  - "我也是在我的美国朋友指点下，一步步添加了 100 多名好友后，最终成功获得了这份工作。"
  - **互助小组**：评论区常有"有没有 linkedin 互加好友的[萌萌哒 R]"，积极参与！
- **主动出击 (Be Proactive!)**
  - 对于你特别心仪的公司和职位，不要只点"一键发送简历"。
  - **发一封简短的 Cover Letter**：几句话说明你的优势、履历，表达你的诚意。"HR 的账号可能每天都有几百封简历涌来，你在这时候做的额外努力，可能更容易引起他/她的关注。"
  - 附上作品集或相关链接（如新闻报道、项目展示）。

### 📝 **简历准备 (Your CV)**

- **中英文版本必备**：尤其是投递中资企业或本地华人公司。
- **边投边完善**：不必追求一次性完美，根据不同职位要求微调。但也有小伙伴表示："我曾试过投不同岗位，去修改简历中适配的部分...事实证明也没多大用。" 所以，核心还是内容的真实和亮点突出。
- **关键词优化**：参考 JD（职位描述），植入相关技能和经验关键词。

### 🧭 **求职渠道大盘点 (Where to Look?)**

- **JobStreet**: 新加坡老牌招聘网站，但很多人的反馈是"一天投几十封，然并卵"，"反馈邮件告诉你，你有几百个竞争者"。竞争异常激烈。
- **MyCareersFuture (MCF)**: 新加坡政府官方求职平台，很多职位需要 SingPass 登录。"这个上面会有很多新加坡政府 po 的相关工作～适合想要去政府工作的小伙伴，但是这个需要 singpass 登录，需要有身份！！！"
- **Indeed**: 兼职信息相对较多，"适合没有时间做全职的姐妹，或者想要半工半读的伙伴！"
- **Glassdoor**: "上面有比较详细的面试问题&面试经验&薪资情况！" 做好面试准备和薪资谈判的必备工具。
- **Glints**: "这个网站的优点就是可以快速定位行业+快速申请，针对已经清晰规划好自己的职业的伙伴～"
- **公司官网 (Company Career Pages)**: 直接投递心仪公司的职位。
- **校友网络 (Alumni Networks)**: "你们 top2 新加坡校友会呢，找内推啊，新加坡内推还是很容易的。"

###🤝 **内推的力量 (The Magic of Referrals)**

"内推"是被反复强调的高效途径！

- **为什么内推重要?** 很多公司优先考虑内推的候选人，成功率远高于海投。
- **如何获取内推?**
  - 通过 LinkedIn 联系目标公司的员工，尤其是校友。
  - 积极参与行业活动，拓展人脉。
  - "尽量在上面认识一些想去的大厂员工，然后争取到内推的机会！"

### 🕵️ **求职中介 (Recruitment Agencies)**

关于中介，大家的看法比较两极。

- **什么时候考虑中介?** 如果自己找了很久没有结果，或者想去一些只通过中介招聘的公司。
- **优点**: 有些中介确实能快速匹配到工作，"不难。我是找的中介，从面试到出去也就一个多星期。"
- **注意事项**:
  - **费用问题**: 了解清楚是向个人收费还是向企业收费。"新加坡本地中介都要收中介费的"，"有些 agent 是公司付费的"。
  - **资质和口碑**: "避雷 58 新加坡中介"，务必选择靠谱中介。
  - **信息来源**: "工作消息基本都在中介手上，因为新加坡雇主好多都是直接和中介对接。"但这可能也分行业和职位。

## 멘 **第三章：修炼内功 —— 应对挑战与心态调整**

找工作不仅是技能的比拼，更是心态的较量。

### ⏳ **做好长期抗战准备 (It's a Marathon, Not a Sprint)**

- **时间跨度长**: 做好心理准备，找工作可能需要几个月甚至大半年。"从刚开始修改简历，到最终找到工作，前后花了大半年时间。"
- **不要轻易放弃**: "就算几个月没收到任何回音，也千万别放弃！不管多难，迈出第一步，坚持走下去，总会有收获的。加油！"

### 📉 **调整期望，先生存再发展 (Manage Expectations)**

- **薪资期望**: 特别是应届生或者没有 PR 的同学，第一份工作的薪资可能不如预期。"4k 对应届生来说根本不低了啊...新加坡就是这样。"
- **职位匹配**: 有时候为了"上岸"，可能需要先接受一个并非最理想的职位。"在新加坡找工就是不断降低自己的预期，直到降无可降，萌生出想回国的心[可怜 R]。"
- **"苟住"策略**: "工资低先上班苟住[捂脸 R]以后想办法跳槽。" 先获得本地工作经验和稳定的签证状态，再图长远发展。

### 🎤 **面试准备 (Ace Your Interviews)**

- **常规面试 (General Interviews)**:
  - 准备好 Behavioral Questions (行为面试题)。
  - 了解公司文化和职位需求。
- **技术面试 (Technical Interviews)**:
  - **SDE (软件开发)**: LeetCode 是基本功 (有小伙伴刷了 LC 650+)，系统设计 (System Design) 越来越重要 (推荐 ByteByteGo)，Java 八股文，处理 Edge Cases 和 Exception 的能力。
  - **Quant Trader (量化交易员)**: 强大的数学建模能力，编程技能 (Python, C++)，概率论、统计推断、金融知识等。
  - **面试语言**: 大部分是英文，但部分中资背景公司可能会有中文面试。
- **警惕面试 PUA**: 有小伙伴分享在中资 Fintech 面试时被 PUA"技术深度不够"。保持自信，清晰表达。

### 🎯 **坚持不懈，相信自己 (Persistence Pays Off)**

- 投递数量要上去："三个月投了近 1000 个岗位"，虽然结果可能不尽如人意，但量变引起质变。
- 多尝试不同方法，不要吊死在一棵树上。

## 💡 **第四章：行业风向与职位聚焦**

不同行业和职位，求职难度和侧重点也有所不同。

### 💻 **软件开发工程师 (SDE/SWE)**

- **求职核心**: LeetCode 刷题，系统设计能力。
- **面试流程**: 通常包括 OA (Online Assessment), Coding 轮 (可能多轮，现场或 take-home), System Design 轮, BQ 轮 (Behavioral Questions)。
- **经验分享**:
  - "某旅游厂...一轮 OA 一轮 Coding 一轮 1.5 小时的系统设计...LC 和 SD 轮都是男子双打。"
  - "某 Web3 厂，惊讶面试竟然全英文...前两面均为来自姐妹 team 的 peer 轮，2 题 medium➕ 一些 theory。"
  - "某中资 Fintech，一面 coding➕java 八股，二面突然变成系统设计...被 PUA 说技术深度不够。"
  - "某 FANG...3 轮 coding 2 题 LC medium...1 轮 SD，1 轮 BQ。"
- **现状**: 竞争激烈，部分基础职位可能 offshore 到成本更低的国家。"AI 的关系吧，已经不大需要这么多程序员了。"

### 💹 **金融交易员 (Trader)**

- **Quant Trader (量化交易员)**:
  - **技能要求**: 极强的数学建模、编程 (Python, C++)、数据结构、算法逻辑。
  - **面试流程**: HR 面 (behavioral) -> Coding Assessment (Python 策略回测) -> Technical Interview (代码、数学、统计、脑筋急转弯、金融知识)。
  - **EP 机会**: 对实力要求很高，但"这个岗完全能发 ep"。
- **Flow Trader (执行交易员)**:
  - **技能要求**: 偏交易执行与流动性管理，市场反应速度，沟通协调。
  - **面试流程**: 投行 S&T Summer Intern 是重要途径，表现好可 fasttrack。Technical 部分比 Quant 容易。
- **经验分享**: 来自"nus 24 毕业 qf 本"的小伙伴详细分享了 QT 和 Flow Trader 的面试经验和岗位特点。

### ✍️ **行政/文员/办公室类 (Admin/Office/Clerical)**

- **求职现状**: Diploma 学历，即使有 3 年 Admin 工作经验，期望薪资 2300 SGD 仍感觉非常困难。
- **竞争激烈**: "看 jobstreet 都是几百人 apply 一个 recruitment [抓狂 R]"，"admin 竞争太大了，太多人申请了"。
- **薪资水平**: 有经验者期望 2.5k 被拒，甚至被压到 2.1k。

## ❗ **第五章：避坑指南 —— 常见误区与特别提醒**

- **盲信官方报告**: MOM 的就业数据和报告仅供参考，个体感受可能差异很大。
- **高估海外学历光环**: "清北本科在国外就业市场等于野鸡本科"，虽然说法夸张，但核心意思是，在新加坡，雇主更看重你的身份、本地经验和实际能力。
- **工作准证是硬伤**: "面试了两个都成功了 但是不够人头弄不到准证 裂[笑哭 R]" 拿到面试甚至口头 Offer，最后卡在准证上的情况屡见不鲜。
- **警惕"幽灵岗"**: "linkedin 很多说在招人的公司挂在那里超过 1-2 年都有了，可是还显示刚推之类的，都假的，没有真的在招人。"
- **中介选择需谨慎**: "避雷 58 新加坡中介，避雷 KPP 公司！黑心中介和公司"，选择有良好口碑和正规资质的中介。
- **做好被"鸽"准备**: "我就被一家本地公司给骗了，12 月两轮面试都过了...结果 1 月中旬告知我 no quota。"
